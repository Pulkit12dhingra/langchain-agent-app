Understanding Basic Arithmetic Operations in Python

This blog post explains how to perform various arithmetic operations using Python, such as addition, subtraction, multiplication, division, integer division, modulus, and exponentiation.

Addition


# Addition
a = 5
b = 3
c = a + b
print(f"Addition: {a} + {b} = {c}")

The output of the code is:

Addition: 5 + 3 = 8

Subtraction


# Subtraction
d = a - b
print(f"Subtraction: {a} - {b} = {d}")

The output of the code is:

Subtraction: 5 - 3 = 2

Multiplication


# Multiplication
e = a * b
print(f"Multiplication: {a} * {b} = {e}")

The output of the code is:

Multiplication: 5 * 3 = 15

Division


# Division
f = a / b
print(f"Division: {a} / {b} = {f}")

The output of the code is:

Division: 5 / 3 = 1.6666666666666667

Integer Division


# Integer Division
g = a // b
print(f"Integer Division: {a} // {b} = {g}")

The output of the code is:

Integer Division: 5 // 3 = 1

Modulus


# Modulus
h = a % b
print(f"Modulus: {a} % {b} = {h}")

The output of the code is:

Modulus: 5 % 3 = 2

Exponentiation


# Exponentiation
i = a ** b  
print(f"Exponentiation: {a} ** {b} = {i}")

The output of the code is:

Exponentiation: 5 ** 3 = 125
